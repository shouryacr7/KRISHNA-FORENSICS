### 🔱 How to Run KRISHNA-FORENSICS
Follow these simple steps to get the tool running on your system:

1. Prerequisites
Make sure you have Python 3.x installed on your computer. You can download it from python.org.

2. Installation
Open your terminal (CMD or PowerShell) and run the following commands:

Bash
# Clone the repository
git clone https://github.com/shouryacr7/KRISHNA-FORENSICS.git

# Enter the project directory
cd KRISHNA-FORENSICS

# Install required dependencies
pip install -r requirements.txt
3. Launching the Tool
On Windows (Easiest Way):
Simply double-click the Run_KRISHNA.bat file. It will automatically check for requirements and launch the tool.

Via Terminal:
If you prefer the command line, use:

Bash
python KRISHNA.py
🔱 Tips for Best Experience:
Run as Administrator: For deep system scans (malware inspection), it is recommended to run the terminal or .bat file as an Administrator.
