# 🛡️ KRISHNA v1.0
**Knowledge-based Real-time Incident & System Hazard Network Analyzer**

KRISHNA is an open-source forensics tool designed to trace malware origins and detect lateral movement in a network.

## 🚀 How to Run
1. Download this folder.
2. Right-click `Run_KRISHNA.bat` and select **Run as Administrator**.
3. Follow the on-screen terminal analysis.
